# Navifare MCP Server

This MCP server wraps the Navifare REST API as MCP tools.

## Tools
- submit_session: creates a price discovery session
- get_session_results: polls a session for results

## Development
- Install: npm i
- Dev: npm run dev
- Build: npm run build
- Start: npm start

## Environment
- NAVIFARE_API_BASE_URL (defaults to https://api.navifare.com/api/v1/price-discovery/flights)

Reference: https://www.stainless.com/mcp/from-rest-api-to-mcp-server
